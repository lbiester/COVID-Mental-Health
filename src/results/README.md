# Results

This directory contains scripts to compute or display results given in our paper. 
Each script is listed along with the figure/table it generates.

* `generate_covid_discussion_graph.py`: Figure 1
* `generate_prophet_graphs.py`: Figures 2, 3, and 4. Also generates graphs for additional metrics.
* `generate_post_count_table.py`: Table 1
* `generate_top_discussion_table.py`: Tables 2 and 3
